## 前言

> Data Structures and Algorithms，简称 DSA，数据结构和算法，数据结构就是计算机中组织和存储数据的方式，而算法就是一步步解决问题的方法。DSA 在语言学习的过程中十分重要，为此我整理了自己学习 DSA 过程中的笔记，其中大部分代码是自己编写的，内容则有些是引用资料，有些是自己撰写的。希望对读者有帮助，也欢迎大家批评指正，谢谢！
